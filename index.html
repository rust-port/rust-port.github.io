Цей сайт посв’ячується грі Rust

Rust - розрахована на багато користувачів комп'ютерна гра в жанрі симулятора виживання, була створена незалежною британською студією Facepunch на чолі якої з 2014 року перебуває Гаррі Ньюмен. Гра вийшла на платформах Microsoft Windows, OS X. Випуск гри відбувся 8 лютого 2018 року.
